# 🌤️ Weather App

Real-time weather app with live data and animations.

## ✨ Features
- 🌡️ Real-time temperature & weather conditions
- 🌙 Day/Night Canvas Animation (stars, shooting stars, clouds)
- 🗺️ Interactive World Map (Leaflet.js)
- 🏙️ Save & manage multiple cities
- 📊 7-day forecast & hourly forecast
- 💨 Air quality, humidity, UV index & more
- ⚙️ Settings page with unit toggle
- 📱 Fully responsive (mobile, tablet, desktop)

## 🛠️ Tech Used
- HTML, CSS, JavaScript
- Canvas API
- Leaflet.js
- WeatherAPI
